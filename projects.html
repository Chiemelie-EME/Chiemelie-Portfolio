Projects

1. Semester Group Website Project

Course: PROG1915 – Project Planning
Instructor: Shriya Bhat
Team: Group 5 – Kang Hong, Edokwe Chiemelie, Rupinderjit Kaur

For this group project, we planned, designed, and developed a fully responsive website as part of our semester coursework. The assignment involved detailed planning using MS Project and JIRA, wireframing, high-fidelity visual design, and final deployment of the website.

I was responsible for content research, image collection, and UI design. This included selecting layout components, color schemes, and ensuring consistency in user experience.

The final deliverable included a working website, project documentation, and a team presentation.

2. Full Semester Planning Using MS Project & JIRA

Assignment: Program Course Planner
Tools: Microsoft Project, JIRA
Scope: All courses, quizzes, deliverables, and final exams (Semester 1)

This individual assignment required building a full semester course schedule using Microsoft Project and JIRA. It included all group and individual assignments, quizzes, and exams across all enrolled courses.

The MS Project file was structured with task dependencies, milestones, and weekly schedules. In JIRA, I broke down each deliverable into sprints, created user stories, and linked tasks with due dates and statuses. This helped develop real-world Agile planning and workload management skills.

3. Personal Portfolio Website Project

Course: PROG1836 – Web Design
Instructor: Mohammad Farooqi

This portfolio website was built to showcase my work, skills, and creativity throughout the Web Design course. It includes eight pages: Home, About Me, Skills, Resume, Projects, Contact, Favorite Links, and a Bonus Feature.

The website is fully responsive and styled using external CSS. A contact form, a resume section, and a YouTube-embedded bonus page were also implemented. This project served as a demonstration of everything I learned in HTML, CSS, and frontend best practices.
